# Landscape

web forum for humans

基于 [Ruby China](https://ruby-china.org) 和 [Homeland](https://github.com/ruby-china/homeland/tree/9adaf2f4c42f9c4f8eb4ed6a5965e417ada6a367) 的论坛应用。

新增的功能：

  * 活动发布与报名
  * 群组内容只对成员可见
  * 文件上传的 AWS S3
  * 更好的多语言支持
  * IPFS 集成，支持将社区中的文章发布到 IPFS 网络
